# 🏗️ Qurilish Mollari Do'koni - Boshqaruv Tizimi

## Talablar
- Python 3.6+ (SQLite o'rnatilgan keladi)

## Ishga tushirish

```bash
cd qurilish-dokoni
python3 server.py
```

Brauzerda oching: **http://localhost:3000**

## Kirish ma'lumotlari
| Username | Parol     | Rol   |
|----------|-----------|-------|
| admin    | admin123  | Admin |

## Funksiyalar
| Bo'lim | Tavsif |
|--------|--------|
| 📊 Dashboard | Bugungi statistika, oxirgi sotuvlar, kam mahsulotlar |
| 💰 Kassa | Sotuv qilish, chek chiqarish, naqd/karta/qarz |
| 📦 Mahsulotlar | Qo'shish, tahrirlash, qidirish, kategoriyalar |
| 🏪 Ombor | Kirim qilish, inventar holati, kirim tarixi |
| 📈 Hisobotlar | Kunlik, oylik, sotuvlar tarixi, xarajatlar |
| 👥 Xodimlar | Admin/Kassir boshqaruvi (faqat admin) |

## Database
SQLite - `dokoni.db` fayl avtomatik yaratiladi.
